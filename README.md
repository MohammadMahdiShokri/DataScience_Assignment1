# 🚗 USA Cars EDA & Samand Web Scraping Project

This repository contains the first assignment for the Applied Data Science course.

## 📋 Project Overview
1. **Exploratory Data Analysis (EDA):** Comprehensive analysis and cleaning of the USA Cars dataset.
2. **Data Visualization:** Using Matplotlib, Seaborn, and Plotly for interactive insights.
3. **Feature Engineering:** Implementing PCA, Mutual Information, and creating new features.
4. **Web Scraping (Bonus):** A Python scraper for extracting Samand car data from Bama.ir.

## 📂 Files in this Repo
* `main_notebook.ipynb`: The primary analysis file.
* `scraper.py`: Python code for the Bama scraper.
* `samand_data.xlsx`: Scraped data from Bama.
* `USA_cars_datasets.csv`: The original dataset used for EDA.
